# 📖 每日待办清单 - 使用指南

## 🚀 快速开始

### 方法一：手动创建（推荐新手）

1. **复制模板**
   - 打开 `Templates/每日待办-简洁版.md`
   - 全选内容 (Ctrl+A)
   - 复制 (Ctrl+C)

2. **创建今日待办**
   - 在你的每日笔记文件夹中新建文件
   - 命名为：`2026-01-09 每日待办.md`（使用今天的日期）
   - 粘贴模板内容 (Ctrl+V)

3. **手动修改日期**
   - 将 `{{date:YYYY-MM-DD}}` 改为今天日期：`2026-01-09`
   - 将 `{{date:MM月DD日 dddd}}` 改为：`01月09日 星期四`

---

### 方法二：使用 Templater 插件（推荐）

如果你已经安装了 Templater 插件：

1. **设置模板文件夹**
   - 打开设置 → Templater → Template folder location
   - 设置为：`Templates`

2. **创建今日待办**
   - 新建笔记
   - 按 `Ctrl+P` 打开命令面板
   - 输入 "Templater: Insert template"
   - 选择 "每日待办-简洁版"
   - 日期会自动填充！

3. **设置每日笔记（可选）**
   - 设置 → 核心插件 → 启用"每日笔记"
   - 设置模板为 `Templates/每日待办-简洁版.md`
   - 每天点击日历图标即可自动创建

## 📝 填写指南

### 1️⃣ 今日聚焦 - 最重要的三件事

**目的**：明确今天必须完成的核心任务

**填写示例**：
```markdown
> **最重要的三件事**
> 1. 完成《程序员的自我修养》第3章阅读并做笔记
> 2. 实现链接器解析符号表的代码
> 3. 整理虚函数相关的3篇笔记到知识库
```

**💡 技巧**：
- 限制在 3 件事以内（多了完不成）
- 必须是具体的、可完成的任务
- 优先级最高，今天一定要做完
- 问自己："如果今天只能做 3 件事，哪 3 件最重要？"

## ⚡ 最佳实践

### ✅ DO（推荐做法）

1. **早晨填写**
   - 一天开始就填写"今日聚焦"和"任务清单"
   - 明确今天的目标

2. **任务具体化**
   - ❌ 差："学习 C++"
   - ✅ 好："阅读《程序员的自我修养》第3章 50 页"

3. **及时打勾**
   - 完成一个任务就立即标记
   - 保持成就感和动力

4. **晚上复盘**
   - 睡前填写"今日记录"部分
   - 总结收获和问题

5. **关联笔记**
   - 将今日学习的内容链接到对应笔记
   - 形成知识网络

### ❌ DON'T（避免的做法）

1. **任务过多**
   - 不要列 20+ 个任务
   - 简洁版建议 8-12 个任务

2. **任务模糊**
   - 避免"学习"、"看书"这种模糊任务
   - 要具体到章节、时间、数量

3. **不回顾**
   - 不要只是列清单，不复盘
   - 反思比执行更重要

4. **完美主义**
   - 完不成不要自责
   - 分析原因，明天改进

## 📞 常见问题

**Q: 忘记填写怎么办？**
A: 没关系，下次记得就好。可以设置每天早上的提醒。

**Q: 任务完不成怎么办？**
A: 分析原因：是任务太多？还是估算错误？调整策略。

**Q: 需要每天都填吗？**
A: 建议工作日/学习日填写，周末可以放松。

**Q: 可以修改模板吗？**
A: 当然！这是你的模板，根据自己的需求定制。
