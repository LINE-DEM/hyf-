# 📋 每日待办清单 - 视觉版

这是一个 Excalidraw 绘图模板，用于创建视觉化的每日待办清单。

## 使用说明

1. 在 Obsidian 中打开此文件
2. 点击右上角的 "更多选项" (三个点)
3. 选择 "Open as Excalidraw drawing"
4. 开始在画布上创建你的视觉化待办清单

## 建议的布局元素

### 核心区域
- 📅 日期标题 (顶部居中)
- 🎯 今日目标 (突出显示的框)
- 📋 任务列表 (分栏布局)

### 任务分类
- 🔴 紧急重要 (左上象限)
- 🟡 重要不紧急 (右上象限)
- 🟢 不重要不紧急 (右下象限)
- 🔵 已完成 (底部)

### 视觉元素
- 使用不同颜色的便签贴
- 添加箭头表示优先级
- 使用复选框图标
- 添加进度条
- 使用 emoji 和图标

### 布局建议
```
┌─────────────────────────────────────────┐
│        📅 2024-01-09 星期二             │
│   🎯 今日目标: [最重要的事]             │
├──────────────┬──────────────┬───────────┤
│  🔴 紧急重要  │  🟡 重要     │ 🟢 其他   │
│  □ 任务1     │  □ 任务4     │ □ 任务7   │
│  □ 任务2     │  □ 任务5     │ □ 任务8   │
│  □ 任务3     │  □ 任务6     │           │
├──────────────┴──────────────┴───────────┤
│  ✅ 已完成任务                          │
│  ✓ 完成的任务1                          │
│  ✓ 完成的任务2                          │
└─────────────────────────────────────────┘
```

## 颜色方案推荐

- 🔴 红色 (#ff6b6b): 高优先级
- 🟡 黄色 (#ffd93d): 中优先级
- 🟢 绿色 (#6bcf7f): 低优先级
- 🔵 蓝色 (#4d96ff): 已完成
- ⚪ 白色 (#ffffff): 背景
- ⬛ 灰色 (#495057): 文字

---

> 💡 **提示**: 在 Excalidraw 编辑模式下，你可以：
> - 使用 Rectangle 工具创建任务卡片
> - 使用 Text 工具添加任务描述
> - 使用 Arrow 工具连接相关任务
> - 使用不同颜色区分优先级
> - 保存为模板以便每日复用
